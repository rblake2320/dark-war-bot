# Dark War Survival Bot

A comprehensive automation bot for Dark War Survival game with an interactive web interface.

## Features

### Website
- **Modern Dark Theme**: Sleek dark interface with purple accent colors
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Interactive Demo**: Try the bot functionality directly in your browser
- **Feature Showcase**: Comprehensive list of all automated features

### Bot Capabilities

The Dark War Survival Bot can automate the following tasks:

#### Resource Management
- **Building Upgrading**: Automatically upgrade buildings when resources are available
- **Resource Gathering**: Collect resources from the map automatically
- **Use Inventory Items**: Manage and use inventory items efficiently

#### Combat & Military
- **Troop Training**: Train troops continuously
- **Heal Troops**: Automatically heal injured troops
- **Attack Monsters**: Hunt monsters for rewards
- **Arena Battles**: Participate in arena for rewards
- **Host Rally**: Organize rally attacks

#### Alliance & Social
- **Alliance Tasks**: Complete alliance tasks automatically
- **Assign Survivors**: Manage survivor assignments
- **Calls**: Respond to alliance calls

#### Progression
- **Research**: Conduct technology research
- **Hero Recruitment**: Recruit and manage heroes
- **Exploration**: Complete exploration missions
- **VIP**: Claim VIP rewards

#### Economy
- **Black Market**: Check and purchase from black market
- **Shop Rewards**: Collect shop rewards
- **Task Rewards**: Claim task rewards
- **Battle Rewards**: Collect battle rewards
- **Mail Rewards**: Collect rewards from mail

#### Protection
- **Shield**: Automatically activate shields when needed

## Multi-Account Support

The bot supports **unlimited accounts** and can:
- Run multiple accounts simultaneously
- Operate 24/7 without breaks
- Manage accounts efficiently with individual configurations
- Monitor all accounts from a single interface

## Getting Started

1. Visit the website at the provided URL
2. Click "TRY DEMO - INTERACTIVE" to test the bot interface
3. Configure which features you want to enable
4. Start the bot and monitor activity in real-time

## Demo Features

The interactive demo includes:
- **Configuration Panel**: Enable/disable specific bot features
- **Bot Control**: Start and stop the bot with visual feedback
- **Activity Log**: Real-time logging of bot actions
- **Status Indicators**: Visual indicators showing bot status

## Technology Stack

- **Frontend**: React 19 + TypeScript
- **Styling**: Tailwind CSS 4 with custom dark theme
- **UI Components**: shadcn/ui component library
- **Routing**: Wouter for client-side routing
- **Build Tool**: Vite for fast development and building

## Notes

This is a demonstration website showcasing the bot's capabilities. The actual bot would be a Windows application that connects to the game client.
